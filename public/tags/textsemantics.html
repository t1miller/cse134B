Text Semantics Tags

The em tag is used for emphasis

The strong tag is used to convey importance and urgency.

The small tag is used for side comments and small print. No animals were harmed in the usage of this tag.

The s tag is used for outdated or irrelevant content. This content is irrelevant.

The cite tag is used for referencing and crediting creative work. Thomas Powell - "Div-itis"

The q tag is used for content quoted from another source. Dumbledore said You shall not pass.

The dfn tag represents the defining instance of a term. A VOD is a Video On Demand

The time element is used to represent time along with its machine-readable form in the datetime attribute. This year.

The code tag represents computer code. public class myClass

The var element represents a variable. myVar

The samp element represents sample output from a program. output

The kbd elemnt represents user input, typically keyboard. Press F to pay respects.

The sub tag is used for subscript and sup is used for superscript. x2 + y3

The i tag is used for an alternate voice or mood. WAZAAAAAAP

The b tag is typically used for bold.

The u tag is typically used for underline.

The mark tag is typicaly used for highlighting.

The span tag is useful when used with global attributes

The br tag is used for a line break.
123 Candycane Lane
North Pole

The wbr tag is used for line break opportunities where a linebreak could be used for text-wrapping purposes, but does not need to be.
